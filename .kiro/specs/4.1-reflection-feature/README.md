# 4.1 振り返り機能

## 概要

目標達成状況の振り返りと改善点を記録する機能。ユーザーが任意のタイミングで目標の進捗を振り返り、総括や改善点を入力できる。

## 目的

- 目標達成状況の定期的な振り返りを促進
- 改善点の明確化と記録
- 過去の振り返りの履歴管理

## 主要機能

1. **振り返り入力画面**: 総括、惜しかったアクション、進まなかったアクション、未着手アクションの入力
2. **振り返りAPI**: CRUD操作（作成、読取、更新、削除）
3. **履歴表示機能**: 過去の振り返り一覧の表示
