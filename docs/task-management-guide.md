# Task Management User Guide

## Overview

The Task Management feature helps you organize and track your daily tasks derived from your mandala chart goals.

## Getting Started

### Accessing Tasks

1. Navigate to the Tasks page from the main menu
2. View your tasks organized by status: Not Started, In Progress, Completed, Skipped

### Task Information

Each task displays:
- **Title**: Brief description of the task
- **Description**: Detailed information (optional)
- **Type**: Execution (one-time) or Habit (recurring)
- **Status**: Current progress state
- **Estimated Time**: Expected duration in minutes
- **Deadline**: Due date (if set)
- **Associated Action**: Parent action from your mandala

## Managing Tasks

### Updating Task Status

1. **Quick Update**: Use the dropdown in the task card
2. **Detailed Update**: Click the task to open detail view

**Status Options:**
- **Not Started**: Task hasn't been begun
- **In Progress**: Currently working on the task
- **Completed**: Task is finished
- **Skipped**: Task won't be completed

### Adding Notes

1. Open task detail page
2. Use the note input field
3. Click "Add Note" to save
4. Edit or delete existing notes as needed

### Viewing History

The task detail page shows:
- Status change history
- Timestamps for all changes
- Progress timeline

## Filtering and Search

### Status Filters

Filter tasks by one or more statuses:
- Check desired status boxes in the filter panel
- Tasks update automatically

### Deadline Filters

- **Today**: Tasks due today
- **This Week**: Tasks due within 7 days
- **Overdue**: Past due incomplete tasks

### Action Filters

Filter by associated actions from your mandala chart.

### Search

- Enter keywords in the search box
- Searches task titles and descriptions
- Results update as you type (300ms delay)

### Saved Views

1. Apply desired filters and search
2. Click "Save View"
3. Enter a name for your view
4. Access saved views from the dropdown

## Bulk Operations

### Selecting Tasks

- Use individual checkboxes on task cards
- Use "Select All" to choose all visible tasks

### Bulk Actions

When tasks are selected:
- **Bulk Complete**: Mark all selected as completed
- **Bulk Delete**: Remove all selected tasks
- **Clear Selection**: Deselect all tasks

## Progress Tracking

### Visual Indicators

- **Progress Bar**: Shows overall completion percentage
- **Color Coding**: 
  - Red (0-33%): Low progress
  - Yellow (34-66%): Medium progress
  - Green (67-100%): High progress

### Status Grouping

Tasks are automatically grouped by status with counts displayed.

## Tips for Effective Task Management

### Daily Workflow

1. Review tasks each morning
2. Update status as you work
3. Add notes for important details
4. Mark completed tasks promptly

### Organization

- Use filters to focus on priority tasks
- Save views for common filter combinations
- Review overdue tasks regularly
- Use bulk operations for efficiency

### Time Management

- Check estimated times when planning
- Use deadline filters for urgent tasks
- Track completion times in notes
- Adjust estimates based on experience

## Troubleshooting

### Common Issues

**Tasks not updating:**
- Check internet connection
- Refresh the page
- Verify you're logged in

**Filters not working:**
- Clear all filters and reapply
- Check if tasks match filter criteria
- Try refreshing the page

**Search not finding tasks:**
- Check spelling
- Try partial keywords
- Clear search and try again

### Getting Help

If you encounter issues:
1. Check this guide first
2. Try refreshing the page
3. Contact support if problems persist

## Keyboard Shortcuts

- `Ctrl/Cmd + F`: Focus search box
- `Escape`: Clear current selection
- `Enter`: Save note (when in note input)

## Mobile Usage

The task management interface is optimized for mobile:
- Swipe gestures for quick actions
- Touch-friendly buttons and controls
- Responsive layout for all screen sizes
